# CodeAssist AI

A powerful CLI-based coding assistant.

## Features
- **Agent Mode**: Full autonomous tool use (Read, Write, Edit, Bash, Glob).
- **Ask Mode**: Quick Q&A.
- **Auto Code Edit**: Precise file modifications.
- **Auto Command Execute**: Runs shell commands.

## Setup
1. `pip install -r requirements.txt`
2. Add your OpenAI API key to `.env`
3. Run: `python main.py`
